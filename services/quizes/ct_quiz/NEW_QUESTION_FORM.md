# Бланк Нового Питання

Це найпростіший формат для роботи зі мною далі.
Ти просто копіюєш цей бланк, заповнюєш його в чаті, а я вже вставляю питання в банк і за потреби доповнюю службові поля.

## Заповнюваний бланк

```md
Клас: 2 / 3 / 4
Тип: core / extension / не знаю

conceptKey: 
conceptLabel: 

difficulty: foundational / developing / advanced / не знаю
progressionBand: recognize / apply / reason / не знаю

Що саме має перевіряти питання:

Текст питання:

Варіанти відповідей:
1. 
2. 
3. 
4. 

Правильна відповідь: 1 / 2 / 3 / 4

Пояснення:

Чим це питання відрізняється від уже наявних:
```

## Як я це оброблятиму

Після твого заповнення я:

1. перевірю, чи не дублює питання існуючі;
2. уточню `conceptKey`, `difficulty` і `progressionBand`, якщо треба;
3. скажу, чи це краще для `core` чи `extension`;
4. додам питання в `questions.js`;
5. оновлю службові множини, якщо потрібно;
6. запущу тести.

## Швидка підказка

- Якщо дитина просто має впізнати ідею, це часто `recognize`.
- Якщо треба використати ідею в новій ситуації, це часто `apply`.
- Якщо треба порівняти, передбачити або зробити висновок, це часто `reason`.

Якщо ти не впевнений у будь-якому полі, сміливо пиши `не знаю` — я допоможу добрати правильне значення.
