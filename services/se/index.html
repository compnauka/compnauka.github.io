Завдання

Завдання 1: Розуміння інтерфейсу

Знайдіть в інтерфейсі пошукової системи:

  • Адресний рядок (де знаходиться?)
  • Пошуковий рядок (у чому різниця?)
  • Кнопку пошуку та "Мені пощастить"

Натискайте на значки "?" для підказок.

Завдання 2: Ключові слова

Спробуйте здійснити пошук за такими запитами:

  • "Столиця України" - простий запит
  • "погода Київ сьогодні" - складений запит
  • "що таке фотосинтез" - запитання
  • "найкращі ресторани Львова недорого" - запит з уточненням

Тепер спробуйте поставити запит, який краще підійде для ШІ-пошуку, наприклад: "Поясни, як працює класичний пошук порівняно з ШІ-пошуком, для п'ятикласника."

Завдання 3: Аналіз результатів

Після пошуку зверніть увагу на:

  • Заголовки результатів (сині тексти)
  • URL-адреси (зелені тексти)
  • Опис сторінки (сірий текст)
  • Підсвічені ключові слова

Якщо б це був ШІ-пошук, як би могла виглядати відповідь на запит "користь зеленого чаю"?

Підказки:

Адресний рядок: Показує веб-адресу (URL) поточної сторінки. Наприклад, https://www.google.com.

Пошуковий рядок: Сюди ви вводите слова (ключові слова), щоб знайти інформацію. Наприклад, рецепт борщу.

Ключові слова: Головні слова, що описують те, що ви шукаєте. Чим точніші ключові слова, тим кращі результати для класичного пошуку.

ШІ-запит: Для ШІ-пошуку можна ставити більш розгорнуті, розмовні запитання.

"Мені пощастить": Кнопка, яка намагається одразу відкрити найімовірніший результат, минаючи сторінку зі списком.

Безпечні дитячі пошукові системи

  • Kiddle — безпечна пошукова система для дітей з фільтрацією контенту.
  • Swiggle — освітня пошукова система для школярів.

Використовуйте ці ресурси для безпечного та цікавого пошуку інформації!
