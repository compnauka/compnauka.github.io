Що таке типи даних?

Типи даних — це способи представлення інформації в комп'ютері. Вони визначають, які значення може зберігати змінна та які операції з нею можна виконувати.

Основні типи даних

Числа

Числові значення використовуються для математичних обчислень.

Цілі числа: 1, 42, 100, -5

Дробові числа: 3.14, 0.5, 2.71

Операції: додавання, віднімання, множення, ділення.

Текст (Рядки)

Рядки використовуються для зберігання текстової інформації.

"Привіт, світ!"

"Я люблю програмування!"

"12345" (це також текст, навіть якщо містить цифри)

Операції: об'єднання, розділення, пошук.

Логічні значення (Булеві)

Булеві значення можуть бути лише true або false.

true

false

Використовуються для умовних операторів та рішень у програмах.

Списки

Списки дозволяють зберігати колекції даних.

[1, 2, 3, 4, 5]

["яблуко", "банан", "апельсин"]

[1, "два", 3, "чотири"]

Операції: додавання, видалення, сортування елементів.
