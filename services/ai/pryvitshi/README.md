# Привіт, ШІ! — документація проєкту

## Призначення

`Привіт, ШІ!` — статична інтерактивна книга українською мовою для дітей про штучний інтелект. Це діалог Тимка з Розумком: користувач переходить між розділами, читає повідомлення-бульбашки та прозову розповідь, переглядає ілюстрації, слухає аудіо та відповідає на квізи.

Ціль — безпечна, доступна, автономна, педагогічно цілісна книга для школи без реєстрації.

## Поточна структура файлів

```text
pryvitshi/
├── index.html          # HTML-каркас, підключення style.css та main.js
├── style.css           # Усі стилі (власні, без зовнішніх фреймворків)
├── main.js             # Оркестратор: навігація, таймери, прогрес
├── bookSections.js     # Весь контент книги: розділи, повідомлення, квізи, медіа
├── renderers.js        # DOM-рендеринг усіх типів повідомлень і блоків
├── quiz.js             # Рендеринг і логіка квізу
├── navigation.js       # Рендеринг бокового меню-навігації
├── menu.js             # Відкриття/закриття мобільного меню
├── storage.js          # Збереження та читання прогресу з sessionStorage
├── validator.js        # Валідація структури bookSections при старті
├── 000.webp … 014.webp # Ілюстрації
└── AI_SONG.mp3         # Аудіофрагмент для розділу про генеративний ШІ
```

## Поточний стан — що вже зроблено

- Tailwind CDN **не підключено** — усі стилі власні, BEM-подібні класи
- Plausible **не підключено**
- Весь рендеринг контенту — через **DOM API** (`createElement`, `textContent`), не `innerHTML`
- `innerHTML` залишено лише для двох випадків: `chatWindow.innerHTML = ''` і `navContainer.innerHTML = ''` (очищення контейнерів)
- Контент у `bookSections.js` — структуровані JS-об'єкти з типізованими блоками, без довільного HTML
- Модульна структура: `main.js` → `renderers.js`, `quiz.js`, `navigation.js`, `menu.js`, `storage.js`, `validator.js`
- Валідатор запускається при старті й виводить помилки в консоль
- `viewport` коректний: `width=device-width, initial-scale=1.0` (zoom не заблоковано)
- Skip-link, `aria-live` через `#a11y-announcer`, `aria-expanded`/`aria-controls` на меню
- Google Analytics залишено (єдина зовнішня залежність з трекінгом)
- Google Fonts підключено через CDN

## Типи повідомлень

| Тип | Опис |

| `cover` | Обкладинка з кнопкою «Розпочати» |
| `ai` | Бульбашка від Розумка |
| `user` | Бульбашка від Тимка |
| `note` | Зелена інфо-картка |
| `narrator` | Проза розповіді без бульбашки, по центру |
| `quiz` | Інтерактивний квіз |
| `activity` | Інтерактивна вправа (`activityType: 'classifier'` \| `'20questions'`) |

## Як запустити локально

```bash
python3 -m http.server 8080
# або
npx serve .
```

Відкрити: `http://localhost:8080/`

ES-модулі вимагають HTTP-сервера — просте відкриття `index.html` через `file://` не спрацює.
