Програма складається з окремих команд, кожна в новому рядку:

вгору    // один крок вгору
вниз     // один крок вниз

вправо   // один крок вправо

Можна використовувати повтори для скорочення запису:

вгору 3   // три кроки вгору
вправо 2  // два кроки вправо
