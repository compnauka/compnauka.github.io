# SERVICE_SHELL_BLUEPRINTS.md

## Призначення

Цей документ дає короткі shell-схеми для кожного сервісу. Його можна використовувати як стартову точку для розробника, агента або генератора коду.

---

# 1. АРТ Текст

## Shell

- Titlebar
- Menu bar
- Toolbar
- Contextbar для тексту / таблиці / зображення
- Document workspace
- Statusbar

## Menu bar

`Файл · Редагування · Вставка · Формат · Перегляд · Допомога`

## Toolbar groups

1. Новий · Відкрити · Зберегти
2. Скасувати · Повернути
3. Стиль · Шрифт · Розмір
4. Ж · К · П · З
5. Вирівнювання
6. Списки · Відступи
7. Колір тексту · Виділення
8. Вставити зображення · Таблицю
9. Zoom

## Workspace

- центр: сторінки документа;
- плаваючі палітри не повинні зсувати сторінку вниз;
- документ або його контейнер повинен мати focus-visible.

## Statusbar

- слова;
- символи;
- мова;
- масштаб.

## Menu bar

`Файл · Редагування · Вставка · Інструменти · Перегляд · Допомога`

## Toolbar groups

1. Новий · Відкрити · Зберегти · Друк
2. Скасувати · Повернути
3. Вибір · Пензлик · Гумка · Заливка · Текст
4. Фігури · Штампи
5. Палітра · Без кольору · Свій колір
6. Товщина · Непрозорість
7. Підкладка
8. Zoom

## Workspace

- центральне полотно;
- без обов'язкових бокових панелей;
- палітри мають накладатися, а не зсувати полотно;
- полотно або його контейнер повинні мати focus-visible;
- для молодших учнів context-параметри не повинні "мигати".

## Statusbar

- інструмент;
- координати;
- колір;
- товщина;
- масштаб.

## Toolbar groups

- масштаб.
