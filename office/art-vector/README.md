# АРТ Вектор

Браузерний векторний графічний редактор для шкільного офісного набору.

## Що реалізовано

- SVG-полотно без реєстрації та без бекенду
- Інструменти: вибір, олівець, лінія, стрілка, прямокутник, еліпс, трикутник, ромб, зірка, текст
- Переміщення, виділення, зміна розміру
- Редагування тексту
- Контур, заливка, товщина, прозорість, розмір шрифту
- Підкладка: немає / сітка / рядки
- Прив'язка до сітки
- Undo / Redo
- Дублювання, видалення, порядок шарів
- Збереження проєкту в JSON
- Експорт у SVG та PNG
- Друк
- Автозбереження чернетки в localStorage

## Гарячі клавіші

- V — вибір
- P — олівець
- L — лінія
- A — стрілка
- R — прямокутник
- O — еліпс
- D — ромб
- S — зірка
- T — текст
- Ctrl+Z / Ctrl+Y — скасувати / повернути
- Ctrl+D — дублювати
- Ctrl+C / Ctrl+V — копіювати / вставити
- Ctrl+S — зберегти проєкт
- Ctrl+Shift+S — PNG
- Delete / Backspace — видалити
