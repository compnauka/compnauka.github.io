# UI_MIGRATION_TO_STANDARD.md — АРТ Вектор

## Поточний стан
Сервіс уже має сильний тулбар, групування інструментів і другий ряд параметрів, але ще не повністю інтегрований у єдину логіку лінійки.

## Основні розриви
1. Немає меню `Вставка`
2. Немає меню `Формат`
3. Другий ряд треба формалізувати як контекстний
4. Треба чітко розвести роль із АРТ Схемами

## План
### P0
- додати `Вставка`;
- додати `Формат`;
- перенести туди частину логіки з другого ряду.

### P1
- зробити другий ряд явною контекстною панеллю;
- уніфікувати статус-бар.

### P2
- описати межу між Вектором і Схемами;
- додати шаблони постерів і діаграм.

## Definition of Done
- меню відповідає загальному каркасу;
- другий ряд — контекст, а не випадковий набір контролів;
- роль сервісу чітко визначена.

## Додатково щодо контекстного UI

- відмовитися від великого другого рядка параметрів під toolbar;
- складні властивості перенести в inspector;
- floating mini-toolbar залишити тільки для коротких швидких дій;
