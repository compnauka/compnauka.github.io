# UI_STANDARD.md — ПЛЮС Малюнки

Сервіс: ПЛЮС Малюнки  
Тип: растровий графічний редактор  
Основна аудиторія: 1–6 класи

## 1. Меню
`Файл · Редагування · Вставка · Інструменти · Перегляд · Допомога`

### 1.1. Розподіл логіки
- `Вставка` — текст, зображення, штампи, готові фігури
- `Інструменти` — вибір, пензлик, гумка, заливка, режими малювання

## 2. Обов'язкові інструменти
- Вибір
- Пензлик
- Гумка
- Заливка
- Фігури
- Штампи
- Текст

## 3. Панель інструментів
Групи:
- Файл
- Історія
- Інструменти
- Вставка швидких об'єктів
- Кольори
- Параметри інструмента
- Масштаб / вигляд

## 4. Контекст / inspector
Для молодшої школи контекстні властивості не повинні бути нестабільними.  
Дозволені варіанти:
- sticky context panel;
- простий inspector-блок;
- відсутність окремого contextbar, якщо параметри вже є на основній панелі.

### 4.1. Що показувати
- малювання: товщина, непрозорість;
- фігура: контур, заливка, товщина;
- текст: шрифт, розмір, колір;
- вибране: розмір, видалення.

## 5. Робоча область
- полотно має бути чистим;
- палітри не повинні зсувати layout;
- панелі не повинні створювати вертикальний шум;
- полотно або контейнер полотна мають мати focus-visible.

## 6. Статус-бар
- інструмент;
- координати;
- розмір полотна;
- підкладка;
- масштаб.

## 7. Гарячі клавіші
Відповідно до `KEYBOARD_SHORTCUTS.md`:
- Delete — видалити вибране
- Esc — закрити overlay або скасувати активний режим
- Ctrl+0 — масштаб 100%

## 8. Заборонено
- приховувати `Вибір`;
- приховувати `Текст`;
- ламати верстку палітрами;
- робити "миготливий" contextbar як основний спосіб керування для молодших учнів.
