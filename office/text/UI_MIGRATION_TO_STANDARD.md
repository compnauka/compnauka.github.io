# UI_MIGRATION_TO_STANDARD.md — ПЛЮС Текст

## Поточний стан
Зараз сервіс має меню `Файл · Редагування · Вставка · Перегляд · Допомога`, сильний тулбар форматування і сторінковий режим.

## Основні розриви
1. Немає меню `Формат`
2. Логіка форматування занадто залежить від тулбара
3. Контекст для таблиці й зображення не формалізований
4. Масштаб має бути у форматі `− 100% +` праворуч

## План
### P0
- додати `Формат`;
- привести toolbar до схеми `Файл → Історія → Формат → Масштаб`;
- уніфікувати масштаб.

### P1
- додати контекстну панель для таблиць;
- додати контекстну панель для зображень;
- уніфікувати select і toggle.

### P2
- уточнити статус-бар;
- уніфікувати модальні вікна.

## Definition of Done
- меню `Формат` існує;
- контекст для таблиць і зображень є;
- масштаб і розташування груп синхронізовані з лінійкою.

## Додатково щодо контекстного UI

- прибрати постійний дублюючий ряд контекстних команд під toolbar;
- для виділеного тексту перейти на inline mini-toolbar;
- для таблиць перейти на context menu або small popover;
- параметри зображення винести в inspector або popover-inspector;
