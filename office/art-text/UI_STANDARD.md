# UI_STANDARD.md — АРТ Текст

Сервіс: АРТ Текст  
Тип: текстовий редактор  
Статус: локальний обов'язковий UI-стандарт

## 1. Роль сервісу

АРТ Текст — це легкий шкільний текстовий редактор для творів, переказів, невеликих рефератів, інструкцій, документів із таблицями й зображеннями.

## 2. Обов'язкова структура меню

`Файл · Редагування · Вставка · Формат · Перегляд · Допомога`

### Файл
- Новий
- Відкрити…
- Зберегти
- Зберегти як…
- TXT / RTF / DOCX
- Друк

### Редагування
- Скасувати
- Повернути
- Вирізати / Копіювати / Вставити
- Знайти…
- Виділити все

### Вставка
- Таблиця…
- Зображення…
- Горизонтальна лінія

### Формат
- Стиль абзацу
- Шрифт
- Розмір
- Жирний / Курсив / Підкреслення / Закреслення
- Вирівнювання
- Списки
- Відступи
- Колір тексту
- Колір виділення

### Перегляд
- Масштаб − / 100% / +
- Портретна / Альбомна

### Допомога
- Клавіатурні скорочення
- Про АРТ Текст

## 3. Основна панель інструментів

Групи:
- Файл
- Історія
- Стиль
- Символьне форматування
- Абзац
- Кольори
- Вставка
- Масштаб

## 4. Контекстна панель

### Для тексту
- базове форматування;
- вирівнювання;
- колір.

### Для таблиці
- рядки / стовпці;
- межі;
- вирівнювання.

### Для зображення
- розмір;
- вирівнювання;
- обтікання;
- видалити.

## 5. Робоча область
- документ подається як окремі сторінки;
- полотно не зсувається вниз через палітри;
- панелі не ламають сторінковий вигляд.

## 6. Статус-бар
- ліва частина: сторінка, слова, символи;
- права частина: масштаб, орієнтація.

## 7. Заборонено
- відсутність меню `Формат`;
- форматування лише через тулбар;
- хаотичний контекст таблиці та зображення.
