# UI_STANDARD.md — АРТ Слайди

Сервіс: АРТ Слайди  
Тип: редактор презентацій

## 1. Меню
`Файл · Редагування · Вставка · Слайд · Перегляд · Допомога`

## 2. Основна панель
- Файл
- Історія
- Слайд
- Вставка
- Шари
- Показ
- Масштаб

Форматування об'єкта не повинно перевантажувати постійну панель.

## 3. Контекстний UI

Патерни:
- текст — `inline mini-toolbar`;
- фігура — `inspector` або `popover`;
- зображення — `inspector` або `popover`.

Форматування об'єктів не повинно перетворюватися на постійний дублюючий ряд під toolbar.

## 3.1. Контекст
- текст: розмір, жирний, вирівнювання, колір;
- фігура: заливка, контур, поворот;
- зображення: розмір, заміна, видалення.

## 4. Робоча область
- ліворуч — компактні мініатюри;
- по центру — активний слайд;
- параметри — контекстно.

## 5. Статус-бар
- слайд X з Y;
- вибраний об'єкт;
- масштаб;
- режим показу.

## 6. Заборонено
- змішувати керування слайдом і форматування в один шумний ряд;
- ховати `Новий слайд` глибше за 1 дію.
