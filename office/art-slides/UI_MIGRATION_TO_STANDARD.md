# UI_MIGRATION_TO_STANDARD.md — АРТ Слайди

## Поточний стан
Сервіс уже має правильний каркас меню й sidebar, але частина об'єктного форматування присутня на постійній панелі.

## Основні розриви
1. Треба скоротити постійну панель
2. Треба перенести більше дій у контекст
3. Треба уніфікувати zoom-контрол

## План
### P0
- лишити на постійній панелі тільки слайд, вставку, показ, шари;
- стандартизувати масштаб.

### P1
- формалізувати контекст для тексту, фігури, зображення;
- перевірити щільність лівої колонки.

### P2
- підготувати макети для типових шкільних презентацій.

## Definition of Done
- основна панель коротка;
- контекстна панель бере на себе форматування;
- thumbnail-sidebar не перевантажує екран.
