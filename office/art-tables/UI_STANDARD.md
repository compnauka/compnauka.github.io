# UI_STANDARD.md — АРТ Таблиці

Сервіс: АРТ Таблиці  
Тип: табличний редактор

## 1. Меню
`Файл · Редагування · Вставка · Формат · Дані · Перегляд · Допомога`

## 2. Панель інструментів
Групи:
- Файл
- Історія
- Форматування
- Вирівнювання і межі
- Кольори
- Формат числа
- Швидкі функції
- Дані
- Масштаб

## 3. Формульний рядок
Формульний рядок — окремий рядок під тулбаром.  
Його не можна переносити в меню або в статус-бар.

## 4. Контекстний UI

Основний патерн: `context menu` або `local popover` для комірки та діапазону.
Для діаграми використовується `inspector`.
Постійний другий ряд під toolbar для таблиць не використовується.

## 4.1. Контекст
- комірка: формат числа, межі, вирівнювання;
- діапазон: статистика, сортування, діаграма;
- діаграма: тип, назва, легенда.

## 5. Статус-бар
- ліва частина: активна комірка, статистика;
- права частина: масштаб, режим.

## 6. Заборонено
- змішувати дані з файловими діями;
- ховати базові функції глибше за 1 рівень.
