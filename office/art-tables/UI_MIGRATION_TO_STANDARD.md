# UI_MIGRATION_TO_STANDARD.md — ПЛЮС Таблиці

## Поточний стан
Сервіс уже майже відповідає стандарту і є найближчим до цільового стану.

## Основні розриви
1. Треба уніфікувати розміри й відступи
2. Треба стандартизувати правий край тулбара
3. Треба формалізувати контекст для діаграми

## План
### P0
- привести zoom до `− 100% +`;
- перевірити порядок груп;
- уніфікувати кнопки та select.

### P1
- формалізувати контекст для діапазону і діаграми;
- уніфікувати статус-бар.

### P2
- підготувати шаблони для навчальних таблиць.

## Definition of Done
- тулбар уніфікований;
- контекст для діаграми описаний;
- сервіс можна використовувати як референс для решти лінійки.

## Додатково щодо контекстного UI

- не додавати постійний другий ряд під toolbar для клітинок і діапазонів;
- реалізувати context menu або local popover для клітинки/діапазону;
- властивості діаграми винести в inspector;
