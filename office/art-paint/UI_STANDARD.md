# UI_STANDARD.md — АРТ Малюнки

Сервіс: АРТ Малюнки  
Тип: растровий графічний редактор

## 1. Меню
`Файл · Редагування · Інструменти · Перегляд · Допомога`

## 2. Обов'язкові інструменти
- Вибір
- Пензлик
- Гумка
- Заливка
- Фігури
- Штампи
- Текст

## 3. Панель інструментів
Групи:
- Файл
- Історія
- Дії
- Інструменти
- Кольори
- Параметри інструмента
- Масштаб / вигляд

## 4. Контекст
- малювання: товщина, непрозорість;
- фігура: контур, заливка, товщина;
- текст: шрифт, розмір, колір;
- вибране: розмір, видалення.

## 5. Робоча область
- полотно має бути чистим;
- палітри не повинні зсувати layout;
- панелі не повинні створювати вертикальний шум.

## 6. Статус-бар
- інструмент;
- координати;
- розмір полотна;
- прив'язка;
- масштаб.

## 7. Заборонено
- приховувати `Вибір`;
- приховувати `Текст`;
- ламати верстку палітрами.
