# Факт чи думка?

Проста браузерна гра для розвитку критичного мислення. Учень читає твердження й обирає, що це: факт чи думка.

## Файли проєкту

- `index.html` — розмітка екранів гри.
- `style.css` — візуальний стиль, адаптивність, фокус-стилі та анімації.
- `script.js` — логіка гри, перемішування тверджень, перевірка відповіді, показ результату.
- `facts.js` — база фактів із джерелами.
- `opinions.js` — база думок, оцінок, забобонів і некоректних узагальнень.
- `SOURCES.md` — короткий аудит джерел і правила їх добору.

## Як запустити

Відкрийте `index.html` у браузері або розмістіть усі файли в одній папці на статичному хостингу.

## Як додати факт

У `facts.js` додайте новий об'єкт:

```js
{
  "id": "fact-unique-id",
  "text": "Твердження, яке можна перевірити.",
  "explanation": "Коротке дитяче пояснення, чому це факт.",
  "sourceTitle": "Назва джерела",
  "sourceUrl": "https://example.com",
  "sourceLanguage": "uk",
  "sourceCheckedAt": "2026-04-30"
}
```

Правила для фактів:

- факт має бути перевірюваним;
- формулювання має бути точним, без перебільшень;
- джерело має відкриватися й підтверджувати саме це твердження;
- для дитячої україномовної гри краще використовувати україномовні джерела;
- якщо якісного україномовного джерела немає, можна залишити авторитетне англомовне.

## Як додати думку

У `opinions.js` додайте новий об'єкт:

```js
{
  "id": "opinion-unique-id",
  "text": "Оціночне або суб'єктивне твердження.",
  "explanation": "Коротке пояснення, чому це думка, смак, припущення або забобон."
}
```

Думка може показувати:

- особистий смак;
- емоційну оцінку;
- забобон або прикмету;
- некоректне узагальнення на кшталт «усі», «завжди», «найкращий».

## Що було оновлено в цій версії

- Додано позначення мови джерела: «українською» або «англійською».
- Замінено кілька старих або проблемних посилань.
- Частину базових фактів переведено на україномовні енциклопедичні джерела.
- Додано `SOURCES.md` із простими правилами перевірки джерел.
- `script.js` залишено простим, без переходу на складну збірку чи фреймворки.

## Мінімальна ручна перевірка перед публікацією

1. Відкрити `index.html`.
2. Натиснути «Почати гру».
3. Перевірити, що у грі 10 питань.
4. Дати кілька правильних і неправильних відповідей.
5. Перевірити, що після факту показується джерело та мова джерела.
6. Перевірити мобільну версію у вузькому вікні браузера.
