# Аудит джерел

Цей файл потрібен, щоб джерела у грі не перетворювалися на випадковий набір посилань. Гра навчає дітей відрізняти факт від думки, тому джерело має бути не просто «десь в інтернеті», а справді підтверджувати твердження.

## Прості правила

1. Для фактів спочатку шукаємо україномовне офіційне, енциклопедичне або навчальне джерело.
2. Якщо якісного україномовного джерела немає, можна залишити авторитетне англомовне: NASA, NOAA, USGS, MedlinePlus, Britannica, університети, офіційні зоопарки або природоохоронні організації.
3. Не використовуємо випадкові блоги, соціальні мережі, SEO-статті та сторінки «цікаві факти», якщо є краща альтернатива.
4. Вікіпедію можна використовувати для простих довідкових фактів, але для важливих тверджень краще мати сильніше джерело.
5. Якщо посилання не відкривається або веде не на те, що треба, його потрібно замінити.

## Що перевірено в цій версії

- Виправлено старі або проблемні URL для Сонця, легень, сов, очей котів і меду.
- Частину базових фактів переведено на україномовні джерела: вода, Земля, Сонце, Місяць, Еверест, океан, Тихий океан, українська абетка, фотосинтез.
- Для всіх фактів додано поле `sourceLanguage` і дату `sourceCheckedAt`.

## Що ще варто зробити пізніше

- Поступово замінити більше англомовних джерел на якісні україномовні.
- Раз на кілька місяців відкривати всі посилання й оновлювати `sourceCheckedAt`.
- Якщо факт формулюється дуже широко, уточнювати його прямо в тексті. Наприклад: «найбільша спекотна пустеля», а не просто «найбільша пустеля».
